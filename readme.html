Follow These steps to set up the wordpress on your local machine

  1. Fork the whole prissolutions folder
  2. Look for prissolutions in the new folder, that is the database sql file
  3. Create a new database in phpmyadmin, name it "prissolutions"
  4. Import the "prissolutions" file into the database you created
  5. That's all, start your server and launch the site in your browser
